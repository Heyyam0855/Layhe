# Ərzaq Səbəti İdarəetmə Sistemi

## Layihə haqqında

Bu layihə işçilərə ərzaq səbəti təqdim edilməsi sisteminin idarə edilməsi üçün nəzərdə tutulmuşdur. Sistem onlayn sifariş, çatdırılma və hesabat funksiyalarını əhatə edir.

## Əsas Xüsusiyyətlər

- ✅ Onlayn sifariş sistemi
- ✅ Avtomatik hesablama və limit yoxlaması
- ✅ Çatdırılma idarəetməsi
- ✅ Hesabat və analitika
- ✅ Keyfiyyət nəzarəti
- ✅ Mobil dəstək

4. **Çatdırılma**: Çatdırılma vaxtını seçin
5. **Qəbul**: Məhsulları qəbul edin və təsdiq edin
