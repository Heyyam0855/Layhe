# Agents Cursor Files

## Layihə Haqqında
Bu qovluq **Cursor AI Code Editor** səhifəsindən yüklənmiş statik veb fayllarını ehtiva edir.

## Fayl Tərkibi
| Fayl Tipi | Say | Təsvir |
|-----------|-----|--------|
| `.css` | 10 | Minified CSS stil faylları |
| `.js.indir` | 8 | Minified JavaScript faylları |
| `css2` | 1 | Google Fonts fayl |

## Texnologiyalar
- **CSS3** — Veb səhifənin vizual dizaynı
- **JavaScript** — Next.js/Webpack ilə paketlənmiş frontend kodu
- **Google Fonts** — Xarici şrift faylı

## Qeyd
Bu fayllar Cursor redaktorunun rəsmi veb səhifəsindən (`cursor.sh`) brauzerlə yüklənmişdir.
Fayllar **minified** (sıxılmış) formatdadır və birbaşa istifadə üçün nəzərdə tutulmayıb.
Öyrənmə məqsədi ilə saxlanılıb.
