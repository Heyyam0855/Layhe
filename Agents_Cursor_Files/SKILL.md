# SKILL — Agents Cursor Files

### Frontend Veb Texnologiyaları
- **CSS3**: Veb səhifə dizaynı, layout, responsive design
- **JavaScript (ES6+)**: Dinamik veb səhifə funksionallığı
- **Webpack**: JavaScript modul paketləmə aləti
- **Next.js**: React əsaslı server-side rendering framework

### Anlayışlar
- Minified kod nədir və niyə istifadə olunur

- Brauzerdə static asset yükləmə prosesi
- Google Fonts inteqrasiyası
